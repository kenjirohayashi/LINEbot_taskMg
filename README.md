<h3>登録</h3>
<ol>
  <li>リッチメニューより登録をタップ。</li>
  <li>LIFFからタスク名を入力して登録。必要に応じて締め切り、通知設定をする。</li>
</ol>
<h3>一覧表示</h3>
<ul>
  <li>リッチメニューより一覧表示をタップ。もしくは@showと送信する。</li>
</ul>
<h3>タスクの消去（完了）</h3>
<ul>
  <li>タスクを一覧表示して、完了をタップ</li>
</ul>
